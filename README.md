# UneOng

O **UneONG** é uma plataforma que possui o objetivo de facilitar a conexão entre **organizações não-governamentais** e **doadores**, com foco na região da cidade de Itu, em São Paulo. Por meio dela, ONGs podem divulgar seus projetos, publicando posts e organizando-os em coleções, enquanto os doadores têm a oportunidade de conhecer novas instituições e encontrar campanhas que desejem contribuir com suas causas.

---

## Como acessar o projeto?

Você precisa ter o node instalado, assim como o npm.

**Baixar as seguintes dependências no terminal do projeto:**

- npm install express sequelize mysql2 ejs ejs-layouts
- npm install --save-dev nodemon
- npm install express
- npm install express ejs
- npm install @supabase/supabase-js
- npm i postgres
- npm install express-session
- npm install cookie-parser
- npm install multer

**Rodar o projeto com um comando no terminal**

npx nodemon app.js
